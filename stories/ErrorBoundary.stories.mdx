# Error Boundary

The `ErrorBoundary` component is used when you wish to handle catching errors within children components and provide a fallback render option.

If an error is thrown within `Children` this compoennt will catch it and redner the `fallback` prop value.

## Use

Import it into a block:

```
import { ErrorBoundary } from '@wpmedia/engine-theme-sdk';


<ErrorBoundary fallback={<div>Fallback</div>}>
  <div>Children</div>
</ErrorBoundary>

```
