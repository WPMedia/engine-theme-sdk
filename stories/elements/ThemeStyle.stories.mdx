# ThemeStyle

A Polymorphic Styled Component wrapper to provide the ability to use theme
settings `font-family`, `color` and `background-color` on an element.

The `ThemeStyle` can be used to render as *any* HTML element by using the
property `element` to denote the HTML element to use. By default the component
will render as a `<div>`.
