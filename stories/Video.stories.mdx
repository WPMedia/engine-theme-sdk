# Video

The `Video` component is generally used to display videos created in [Video Center](https://corecomponents.arcpublishing.com/alc/arc-products/videocenter/).

## Requirement

The playerRoot and videoOrg Fusion environment variables must be defined in order to load the POWA player's library.


## Use

Import it into a block: 

```
import { Video } from '@wpmedia/engine-theme-sdk';

...

<Video uuid={video._id} org="corecomponents" env="prod" />

...

```
