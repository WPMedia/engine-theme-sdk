# ArcLogo

The `ArcLogo` component is generally used for the navigation header block, footer block or anywhere that requires a fallback logo for branding.

## Use

Import it into a block: 

```
import { ArcLogo } from '@wpmedia/engine-theme-sdk';

...

<ArcLogo />

...

```
