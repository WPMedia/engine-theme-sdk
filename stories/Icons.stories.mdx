# Icon

The `Icon` components reference customizable svg images. There are several icons available and can be configured to different colors, sizes and metadata.

## Use

Import it into a block:

```
import { InstagramIcon } from '@wpmedia/engine-theme-sdk';

...

<InstagramIcon />

...

```
