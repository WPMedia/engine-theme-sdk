# Lightbox

The `Lightbox` component displays a set of images by filling the screen, and dimming out the rest of the web page.

## Use

Import it into a block: 

```
import { Lightbox } from '@wpmedia/engine-theme-sdk';

...

<Lightbox />

...

```
